What is Multi-Criteria Analysis?

Multi-criteria analysis (MCA) in GIS is a technique that evaluates several spatial factors simultaneously. It combines a diversity of criteria using weighted analysis to be able to create comprehensive maps, including factors like cost, accessibility, environmental impact, and resource availability within a one analytical framework.

An example of how this works would be if you are trying to find the best location to build a new hospital. You can't just look at one thing: you need land that's flat enough to build on, close to roads, away from flood zones, not on protected land, and accessible to the largest number of people. MCA is the method that takes all of those separate GIS layers and combines them into one map to tell you where the best location is based on all the known information.

To perform a multi-criteria analysis in GIS, one must assign weights or importance to each criterion that will be used and then normalize the data so that all inputs are on the same scale.

Why is MCA Valuable in GIS?

This technique is extremely valuable because most real world problems rely on many factors. MCA provides a structural framework for handling these real-world complexities. For example, it forces you to state how important each factor is relative to others instead of relying on a subjective gut feeling, and computes responses correspondingly. This proves extremely valuable for solving complex issues.

Some concrete ways in which MCA is valuable when mapping these complex issues is that it organises the data in different thematic layers which can be given numerical values. This allows us to visualize the different criteria rather then juggling separate maps.

Furthermore, MCA uses suitability analysis: a single raster layer where scores areas based on if they meet the combined criteria. Weighting is also crucial as it allows us to consider the fact that all criteria are not equally important. For example, if flood risk is twice as important as slope in a given decision, it is assigned twice the weight, and this is applied consistently across every single location in the dataset.

Examples of Questions MCA Can Help Answer

  • Where is the best location for something?
  • How suitable is a given area for a given use?
  • How should resources be allocated across a space?
  • Where should conservation efforts be prioritized or implemented?
  • Which areas will be most vulnerable to climate impacts?
